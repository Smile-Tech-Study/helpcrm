## Путь заявки во Flow

1. **Заявка поступает из У2035 во Flow** после одобрения

2. **Система определяет группу шаблонов** -- по возрасту (до 18 / 18+) и формату программы (онлайн / офлайн).

   :::tip 

   [Группы шаблонов](./../../../Organization/Shablony/_index) должны быть предварительно подготовлены образовательной организацией. Договор, заявление и согласие будут сгенерированы по нужному шаблону

   :::

3. **Ученик в ЛК Flow** проверяет и дополняет свои данные, а также данные законного представителя (если нет 18 лет), подписывает договор, заявление на зачисление и согласие на обработку персональных данных

4. **Провайдер проверяет данные** и одобряет документы на зачисление

5. **Ученик получает доступ в Odin** -- кнопка входа появляется в ЛК Flow (и дублируется письмо-приглашение в Odin) в день старта обучения **при наличии приказа на зачисление в заявке**.  Если приказа пока нет, то ученика можно добавить в Odin по кнопке «Зарегистрировать» на странице заявки и параллельно с обучением продолжить собирать документы на зачисление

6. **После каждого модуля** Flow получает результат из Odin. Провайдер может сгенерировать сертификат по образцу своей организации (Образец сертификата предоставляется в поддержку заранее).

7. **После завершения всех модулей** данные по итоговой аттестации отправляются в У2035, во Flow оформляется приказ об отчислении с успешным/неуспешным завершением

## Какие данные собираются во Flow

### Данные об участнике

#### Поступают автоматически с У2035:

-  ФИО ученика

-  Дата рождения

-  СНИЛС

-  Электронная почта

-  Телефон (можно исправить во Flow на первом шаге)

#### Собираются через ЛК Flow

-  Данные паспорта (серия, номер, дата выдачи, кем выдан, код подразделения) -- для учеников старше 14 лет

-  Данные свидетельства о рождении -- для учеников младше 14 лет

-  Адрес регистрации и фактического проживания

-  Школа и класс

-  Почтовый адрес

-  Фамилия (в дательном падеже)

-  Имя (в дательном падеже)

-  Отчество (в дательном падеже)

-  Требуются специальные условия обучения в связи с ограниченными возможностями здоровья: **Да/Нет**

### Данные законного представителя

Заполняются  в ЛК Flow для участников **младше 18 лет**. (Для совершеннолетних эти поля автоматически заполняются нулевыми значениями.)

-  Степень родства законного представителя

-  ФИО законного представителя

-  СНИЛС законного представителя

-  Дата рождения законного представителя

-  Пол законного представителя

-  Серия паспорта законного представителя

-  Номер паспорта законного представителя

-  Дата выдачи паспорта законного представителя

-  Кем выдан паспорт законного представителя

-  Код подразделения в паспорте законного представителя

-  Регион регистрации законного представителя

-  Населенный пункт регистрации законного представителя

-  Индекс регистрации законного представителя

-  Дом регистрации законного представителя

-  Улица регистрации законного представителя

-  Квартира регистрации законного представителя

-  Регион фактического проживания законного представителя

-  Населенный пункт фактического проживания законного представителя

-  Индекс фактического проживания законного представителя

-  Улица фактического проживания законного представителя

-  Дом фактического проживания законного представителя

-  Квартира фактического проживания законного представителя

-  Фамилия (в дательном падеже) законного представителя

-  Имя (в дательном падеже) законного представителя

-  Отчество (в дательном падеже) законного представителя

-  Телефон законного представителя

-  Email законного представителя

## Данные из У2035 (статусы и идентификаторы)

Обновляются автоматически через API У2035:

-  UntiId (после получения не обновляется)

-  Статус трансферта

-  Результаты ВИ (после получения не обновляется)

-  Дата автоотклонения на У2035 ((после получения не обновляется))

-  Номер заявки на У2035 (после получения не обновляется)

-  Статус заявки на У2035

-  Текущий номер модуля в У2035

-  Даты сдачи ПА

Подробнее о собираемы данных в «[Доп. поля и внешние источники](./../programmy-i-potoki/trebuemye-dlya-kb-dop-polya-i-vneshnie-istochniki)»
